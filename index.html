Chained Hash Table

This is my Data Structures and Algorithms (C++) 2018-2019 Fall term project. We were expected to write a C++ console application which reads a .csv text file that consists of 1.6M tweets and then displays the top 10 most frequent words in the tweets. We had to exclude stop words which is used very often in daily life (i.e. a, an, the, like). 30% of the project’s grade was given for the execution time, so I had to implement the program with a minimal time complexity.

Abode Coin
(Crypto-Currency)

This is a personal project of mine. As the new technologies and solutions arise, I try to keep up with them. With the popularity of blockchain and crypto-currencies, I attempted to create my own crypto-currency. They can be mined and verified on a LAN with at least two computers, send and receive ABCs. I neither created nodes on the internet for everyone to use nor defined any IP in my code. For now, it is a closed network.

It is actually based on an earlier version of Bitcoin, but Abode Coin runs on a totally different network without interfering with any other crypto-currency.
